# WEATHER-DETECTOR

This project is used to detect the temp and humidity of the environment .By using the arduino uno and dht22 sensor 

##Components
  Arduino Uno
  DHT22 sensor
  Jumper wire

#Aim
  By the help of DHt22 sensor to detect the humidity and temperature
